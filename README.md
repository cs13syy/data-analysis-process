# Data Analysis Process (데이터 분석 프로세스)
**가설 수립 -> 데이터 탐색 -> 모델링 -> 결과 도출 및 평가**의 아름답고 평화로운 데이터 분석을 기대했으나, 곧 몇 개의 프로젝트를 진행하면서 그 기대를 완전히 버렸습니다. 분석가의 엄청난 육체적, 정신적 고통이 뒤따르는 데이터 분석 과정에서 제가 했던 고민들, 그 고민들에 대한 해답들을 정리해보았습니다.
